программа №1 перемещает все файлы из папки загрузок (C:\Users\UC33_9\Downloads) в папку с именем компьютера, создаваемую программой в этой папке.
программа №2 выводит номер компьютера, за которым вы находитесь. требуется, если вы не можете понять в какую папку перенеслись файлы.
	#в любом случае можно рассортировать по дате изменения, и будет видно какая папка была изменена последней.

НАСТРОЙКА ПРОГРАММЫ:
	введите в файл INPUT.TXT "y" или "n" для указания замены переносимых файлов, при идентичном названии.
	
	если файл INPUT.TXT окажется пустым, туда по умолчанию введётся "n", то есть отказ от замены.

после завершения выполнения программы №1, вся информация насчёт выполнения команд заносится в файлы, находящиеся в папке "logs/".

в файл %номер компьютера%_log-imports.txt записывается информация в процессе переноса файлов в программе №1.
в файл %номер компьютера%_log-errors.txt записываются ошибки в процессе выполнения этой программы.
